Enerwise

A real-time power monitoring interface for hardware managing solar energy systems. It optimizes fuel and grid costs by keeping power on the solar system when consumption is below 2000W.

Key Features:

  • Smart Load Switching: Automatically toggles to grid power if consumption exceeds the threshold.
  • Automated Backup: Triggers relay modules to safely start gasoline generators during grid outages.
  • Emergency Alerts: Provides instant system status notifications to the user.

CPU SIMULATOR with JS

An interactive CPU Simulator built entirely with JavaScript to visually demonstrate the core Fetch-Decode-Execute (FDE) life cycle of a processor.

Key Features:

  • Live System Bus: Real-time tracking of Address, Data, and Control bus activity during execution.
  • Advanced Addressing: Supports Immediate, Direct, Register-indirect, and Indexed addressing modes.
  • Interactive Debugging: Allows users to edit instruction sequences in memory and step through execution phase-by-phase.
